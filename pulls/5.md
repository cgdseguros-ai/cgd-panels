## Requisito adicional — campos por competência + novo campo LEAD DA OPERADORA

A partir da **competência abril/2026** (`2026-04`), os seguintes campos devem ser utilizados para todos os negócios:

| Campo | Constante | UF |
|---|---|---|
| Operadora | `OPERATOR_FIELD` | `UF_CRM_1771388467` |
| Fonte | `SOURCE_FIELD` | `UF_CRM_1767283877974` |
| Lead da Operadora | `LEAD_OPERADORA_FIELD` | `UF_CRM_1771388342` |

### Regra de data de corte

Adicionar constante:
```js
const FIELDS_V2_SINCE = "2026-04"; // competências >= esta usam os novos campos
```

Em `mapDealToItem`, verificar a competência do negócio:
- Se `competence >= "2026-04"` → usar `OPERATOR_FIELD` (`UF_CRM_1771388467`), `SOURCE_FIELD` (`UF_CRM_1767283877974`) e `LEAD_OPERADORA_FIELD` (`UF_CRM_1771388342`)
- Se `competence < "2026-04"` (negócios mais antigos) → manter o comportamento anterior: `OPERATOR_FIELD` antigo (`UF_CRM_1731877451385`) e `SOURCE_ID` nativo

Para implementar:
1. Manter as duas constantes de operadora:
   ```js
   const OPERATOR_FIELD_OLD = "UF_CRM_1731877451385"; // competências até 2026-03
   const OPERATOR_FIELD     = "UF_CRM_1771388467";    // competências >= 2026-04
   const SOURCE_FIELD       = "UF_CRM_1767283877974"; // competências >= 2026-04
   const LEAD_OPERADORA_FIELD = "UF_CRM_1771388342";  // competências >= 2026-04
   ```
2. Em `fetchDeals`, incluir **todos** os campos no `select`: `OPERATOR_FIELD_OLD`, `OPERATOR_FIELD`, `SOURCE_FIELD`, `LEAD_OPERADORA_FIELD`, `SOURCE_ID`.
3. Em `mapDealToItem`, escolher qual campo usar baseado na competência do negócio (campo `CLOSEDATE_FIELD`).
4. Construir mapa para `OPERATOR_FIELD` (novo) — `loadOperatorMap()` já existente, mas criar também `loadOperatorMapOld()` para `OPERATOR_FIELD_OLD`.
5. Construir mapa para `SOURCE_FIELD` via `crm.deal.fields` → `items`.
6. Construir mapa para `LEAD_OPERADORA_FIELD` via `crm.deal.fields` → `items`.

### Exibição do campo LEAD DA OPERADORA

- Adicionar campo `leadOperadora` no objeto retornado por `mapDealToItem`.
- Exibir "Lead da Operadora" nas tabelas dos modais (LISTA, CONCLUÍDAS, ANÁLISE) como coluna adicional.
- Nos filtros do modal ANÁLISE, adicionar filtro por **Lead da Operadora** além de Operadora e Fonte.
